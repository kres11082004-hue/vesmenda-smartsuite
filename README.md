## 🚀 Deployment to Render

This project is optimized for deployment as a **Static Site** on [Render](https://render.com).

### Easy Deployment (Blueprint)
The included `render.yaml` file allows for automatic configuration.
1. Connect your GitHub repository to Render.
2. Render will automatically detect the `render.yaml` and configure:
   - **Build Command**: `npm install && npm run build`
   - **Publish Directory**: `dist`
   - **Routes**: Automatic index.html rewrite for SPA support.

### Manual Configuration
If not using the Blueprint:
- **Service Type**: Static Site
- **Build Command**: `npm run build`
- **Publish Directory**: `dist`
- **Redirects**: Add a rewrite rule for `/*` to `/index.html` to support React Router.
